# Pac-Man Word Hunter - English Learning Game

A fun Pac-Man themed browser game designed to help 8-year-old children learn English words through an interactive ghost-catching adventure.

## 🎯 About the Game

**Pac-Man Word Hunter** combines the classic arcade fun of Pac-Man with educational English learning. Players control a yellow Pac-Man character through a maze, hunting for ghosts that display English words. The game features Traditional Chinese translations and voice pronunciation to help children learn vocabulary in an engaging, game-based environment.

## 🎮 Game Features

- **Ghost Characters**: Each ghost displays an English word that children need to learn
- **Target Word Display**: Central banner shows the target English word and its Chinese translation
- **Voice Pronunciation**: The target word is pronounced repeatedly using text-to-speech
- **Background Music**: Cheerful background music to enhance the gaming experience
- **Score System**: Earn points by catching the correct ghost, lose points for wrong ones
- **Custom Word Lists**: Parents can input custom English words for dictation practice
- **Responsive Controls**: Supports keyboard, mouse, and touch controls

## 🎯 How to Play

1. **Start the Game**: Click "Start Game" to begin
2. **Find the Target**: Look at the central banner to see which English word you need to find
3. **Catch the Ghost**: Move your green character to catch the ghost with the target word
4. **Score Points**: 
   - +10 points for catching the correct ghost
   - -5 points for catching the wrong ghost
5. **Learn New Words**: Each successful catch introduces a new target word

## 🎮 Controls

### Keyboard
- **Arrow Keys** or **WASD**: Move the player character
- **Mouse**: Move cursor to guide the player
- **Touch**: Touch and drag on mobile devices

### Game Controls
- **Start/Restart**: Begin a new game
- **Pause/Resume**: Pause the game temporarily
- **Sound Toggle**: Turn sound on/off

## ⚙️ Settings

### Custom Words
Parents can add custom word pairs for targeted learning:

1. Click the ⚙️ Settings button
2. Enter words in the format: `english:chinese`
3. One word pair per line
4. Click "Save Words" to use your custom list

**Example:**
```
apple:蘋果
banana:香蕉
cat:貓
dog:狗
```

### Game Settings
- **Background Music**: Toggle background music on/off
- **Word Pronunciation**: Toggle text-to-speech on/off
